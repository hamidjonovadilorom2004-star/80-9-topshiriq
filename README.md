# 80-8-topshiriq
# Supply Chain Logistics Blockchain (8-chi topshiriq)

Bu loyiha mahsulotlarni ishlab chiqaruvchidan sotuvchigacha bo'lgan butun yo'lini (Logistika) blokcheynda kuzatish va audit qilish uchun mo'ljallangan.

## Funksiyalar:
*   **Cargo Registration**: Mahsulotni tizimga birinchi marta kiritish.
*   **Real-time Tracking**: Mahsulotning qayerdaligini va holatini (statusini) blokcheynda yangilab borish.
*   **Ownership Transfer**: Mahsulot egasini o'zgartirish (Ishlab chiqaruvchi → Distribyutor → Sotuvchi).
*   **Audit Trail**: Mahsulotning butun yo'lini (audit) tekshirish.
*   **Authenticity Check**: Mahsulotning haqiqiyligini blockchain orqali tekshirish.

## Steklar:
- **Solidity**: Logistika mantiqi va audit zanjiri uchun.
- **React/Vite**: Admin va kuzatuv paneli uchun.
- **Space Grotesk & Amber Theme**: Logistika sohasiga mos industrial dizayn.

## Ishga tushirish:
1. `cd 8_chi_topshiriq`
2. `npm install`
3. `npm run dev`
